Run 10 ue:
```bash
sudo ./packetrusher multi-ue -n 10
```

Setting 10 ue with wait start interval (500ms)
- note: interval should than 100
- default: interval = 500

```bash
sudo ./packetrusher multi-ue -n 10 -tr 500
```
